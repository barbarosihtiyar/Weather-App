# Weather App
*Weather App is a Javascript web app. In this Weather Application, it offers users the opportunity to search for the city they want and learn the weather conditions.
*

# Features
- Search country
- Search city
- See humidty
- See wind
- 7 country capital

#### In Addition
- Weather description
- Country of the searched city
